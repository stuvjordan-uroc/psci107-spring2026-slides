Definition of an "Agenda"

An agenda is a set of possible locations in the model to which policy could be set.

Definition of a "Voting Rule"
Given a spatial model of policy, a set of persons with single-peaked preferences, and a specified ideal point for each person, and an agenda...
A voting rule is:
  • A set of possible messages from which each person may choose.
  • A mapping that takes each possible profile of messages (one message from each person from her set of available messages) to a policy from the agenda that will be adopted.
Example: Majority Rule with a Binary Agenda
Take a spatial model of policy, an odd number of persons with single-peaked preferences, a specified ideal point for each person, and an agenda with exactly two policies.
Sets of Available Messages
Each person's set of available messages is $\left\{``W",``Y"\right\}$
Mapping
Given any profile of messages (e.g. Person A sends $``W"$, Person B sends $``Y"$, Person C sends $``W"$, ..., Person G sends $``Y"$), the policy adopted is the policy named in the majority of messages.
Example: Two-Thirds Rule with a Binary Agenda
Take a spatial model of policy, an odd number of persons with single-peaked preferences, a specified ideal point for each person, and an agenda with exactly two policies.
Sets of Available Messages
Each person's set of available messages is $\left\{``W",``Y"\right\}$
Mapping
Given any profile of messages the policy adopted is:
  • $W$ if fewer than two-thirds of the persons send message $``Y"$
  • $Y$ if two-thirds or more of the persons send message $``Y"$
Example: Unanimity Rule with a Binary Agenda
Take a spatial model of policy, an odd number of persons with single-peaked preferences, a specified ideal point for each person, and an agenda with exactly two policies.
Sets of Available Messages
Each person's set of available messages is $\left\{``W",``Y"\right\}$
Mapping
Given any profile of messages the policy adopted is:
  • $W$ if fewer than all of the persons send message $``Y"$
  • $Y$ if all of the persons send message $``Y"$
Example: Plurality Rule with Run-offs
Sets of Available Messages
Each person's set of available messages consists of all possible mappings that take each distinct subset of policies to a "top choice" from that subset
Mapping
  • Take the persons' top choices from the set of all policies. If there is a single policy that is the top choice for the largest number of the persons, adopt that policy and stop. Otherwise...
  • Take the policies that are tied for first place w.r.t. to the number of persons naming that policy as a top choice. If there is a single policy that is named "top choice" by the largest number of persons from that set, adopt that policy and stop. Otherwise...
  • Take the policies that are tied for first place w.r.t. to the number of persons naming that policy as a top choice from the set in the previous step. If there is a single policy that is named "top choice" by the largest number of persons from that set, adopt that policy and stop. Otherwise...
  • Continue as above until you have a winner!
