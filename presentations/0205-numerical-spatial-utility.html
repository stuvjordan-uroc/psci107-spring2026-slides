Utility and Utility Functions

Utility Representation of a Preference
Take any set of alternatives. (e.g. $A$, $B$, $C$, $D$, etc.)
A utility representation of a person's preferences over that set assigns a number $U(x)$ to each alternative $x$.
Given any pair of alternatives $x$ and $y$...
  • $U(x) > U(y)$ means "the person prefers $x$ to $y$"
  • $U(y) > U(x)$ means "the person prefers $y$ to $x$"
  • $U(x) = U(y)$ means "the person is indifferent between $x$ and $y$"
Example
Construct a utility representation of your preference ordering over the following Beyonce albums: Lemonade, Cowboy Carter, Dangerously In Love, Renaissance
Here's my ranking (favorite to least-favorite): Lemonade, Cowboy Carter, Dangerously In Love, Renaissance
So I'll go with the following
  • $U(\text{lemonade}) = 403$
  • $U(\text{cowboy carter}) = 6$
  • $U(\text{renaissance}) = -32$
  • $U(\text{dangerously in love}) = -33$
Important Notes
  • Negative vs. positive doesn't mean anything. (Negative isn't "extra special bad").
  • All that matters is ordering. (We will need to clarify this later!)
  • You get the same "meaning" from any transformation that preserves ordering, such as adding a the same constant to every value.

Utility function
Take any set of alternatives. (e.g. $A$, $B$, $C$, $D$, etc.)
Suppose that every alternative has a defined value on each of a set of characteristics.
A utility function assigns a utility level to any generic alternative as a function of the that alternative's value on the characteristics.
Example
Construct a utility function to represent your preferences over playlists.
(To do this, you come up with one or more characteristics that any given playlist could have. Then you specify a function that assigns a utility value to any given playlist on the basis of its values with respect to those characteristics.)
Some characteristics of playlists
  • Number of Frank Ocean songs
  • Whether it includes anything by Bob Seger
Utility level $$ \text{number of Frank Ocean songs} + \begin{cases} -1,000,000,000,000 & \text{if any Bob Seger songs} \\ 0 & \text{otherwise} \end{cases} $$
A utility function is nothing fancy. It's just a way to...
  • ...produce a utility representation of preferences over any arbitrary subset of a set of alternatives.
  • ...depict the relationship between the characteristics of a thing and that thing's "preferability" to a person.
Example: How utility functions let us model the way a person resolves conflicts between values

Basic Numerical Spatial Model

Definition: The Numerical Spatial Model
Take any set of policies differentiated by their relative degrees of something-ness.
Assign a number to each policy, with higher number assigned to policies with more something-ness!
That's it!!! Now it's a numerical spatial model!

The tent shaped utility function

Definition: The Tent Shaped Utility Function
...models a person's preferences over policies in a numerical spatial model.
Let $x$ be the something-ness of any given policy....
Let $p$ be the person's ideal level of something-ness...
